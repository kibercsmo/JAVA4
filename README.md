# Реализация консольной реализации управления библиотекой книг

## Как работает

* lib.addbook() - добавляет книгу в библиотеку

* lib.findBookById() - ищет книгу по id, если нашел, то выводит описание, если нет то null

* lib.findBookById() - выводит все книгу по автору

* lib.borrowBook() - меняет статус книги на false

* lib.returnBook() - меняет статус книги на true

* lib.getAvailableBooks() - выводит все книги со статусом true

* lib.printOperationLog() - выводит историю операция по добавлению и изменению статуса книг
